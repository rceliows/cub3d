# MiniLibX Functions Summary

----------------------------------------------------------------------------------------------------------------------------------------------

## Window Management

**mlx_clear_window**
- Parameters: `t_xvar *xvar`, `t_win_list *win`
- Description: Clears the entire window content

**mlx_destroy_window**
- Parameters: `t_xvar *xvar`, `t_win_list *win`
- Description: Destroys a window, removes it from the window list, frees associated graphics context and memory

----------------------------------------------------------------------------------------------------------------------------------------------

## Display Management

**mlx_destroy_display**
- Parameters: `t_xvar *xvar`
- Description: Closes the X display connection

**mlx_init**
- Parameters: None
- Returns: `void*` (pointer to t_xvar)
- Description: Initializes the MiniLibX library, opens X display connection, sets up default screen, colormap, visual settings, and shared memory handling

----------------------------------------------------------------------------------------------------------------------------------------------

## Image Management

**mlx_destroy_image**
- Parameters: `t_xvar *xvar`, `t_img *img`
- Description: Destroys an image, detaches shared memory if applicable, frees the X image, pixmap, graphics context, and image structure

**mlx_get_data_addr**
- Parameters: `t_img *img`, `int *bits_per_pixel`, `int *size_line`, `int *endian`
- Returns: `char*` (pointer to image data)
- Description: Retrieves the memory address of the image data along with its properties (bits per pixel, line size, and endianness)

----------------------------------------------------------------------------------------------------------------------------------------------

## Color Management

**mlx_get_color_value**
- Parameters: `t_xvar *xvar`, `int color`
- Returns: `int`
- Description: Converts a color value to the appropriate format for the current display depth

**mlx_int_get_good_color**
- Parameters: `t_xvar *xvar`, `int color`
- Returns: `int`
- Description: Internal function that performs color conversion based on display depth and RGB bit shifting

----------------------------------------------------------------------------------------------------------------------------------------------

## Event Handling

**mlx_hook**
- Parameters: `t_win_list *win`, `int x_event`, `int x_mask`, `int (*funct)()`, `void *param`
- Description: Registers a generic event hook for any X11 event type with specified event mask

**mlx_expose_hook**
- Parameters: `t_win_list *win`, `int (*funct)()`, `void *param`
- Description: Registers a callback function for window expose events (when window needs redrawing)

**mlx_flush_event**
- Parameters: `t_xvar *xvar`
- Description: Flushes all pending events from the event queue without processing them

----------------------------------------------------------------------------------------------------------------------------------------------

## Keyboard Control

**mlx_do_key_autorepeatoff**
- Parameters: `t_xvar *xvar`
- Description: Disables automatic key repeat for the display

**mlx_do_key_autorepeaton**
- Parameters: `t_xvar *xvar`
- Description: Enables automatic key repeat for the display

----------------------------------------------------------------------------------------------------------------------------------------------

## Display Synchronization

**mlx_do_sync**
- Parameters: `t_xvar *xvar`
- Description: Forces synchronization with the X server, ensuring all previous requests are processed

----------------------------------------------------------------------------------------------------------------------------------------------

## Fullscreen Extension

**mlx_ext_fullscreen**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `int fullscreen`
- Description: Toggles fullscreen mode using XRandR extension, finds appropriate screen resolution, grabs keyboard input in fullscreen mode

----------------------------------------------------------------------------------------------------------------------------------------------

## Internal Helper Functions

**mlx_int_anti_resize_win**
- Parameters: `t_xvar *xvar`, `Window win`, `int w`, `int h`
- Description: Prevents window resizing by setting minimum and maximum size hints to the same dimensions

**mlx_int_do_nothing**
- Parameters: `void *param`
- Description: Empty placeholder function that does nothing

**mlx_int_get_visual**
- Parameters: `t_xvar *xvar`
- Returns: `int` (0 on success, -1 on failure)
- Description: Attempts to find a TrueColor visual for the display, sets up private colormap if needed

**mlx_int_deal_shm**
- Parameters: `t_xvar *xvar`
- Returns: `int`
- Description: Checks and configures shared memory (XShm) extension availability for the display connection

**mlx_int_rgb_conversion**
- Parameters: `t_xvar *xvar`
- Returns: `int`
- Description: Calculates bit shift values for RGB color conversion based on the visual's color masks

**mlx_int_set_win_event_mask**
- Parameters: `t_xvar *xvar`
- Description: Updates the event mask for all windows based on registered hooks

----------------------------------------------------------------------------------------------------------------------------------------------

## Event Parameter Handlers

**mlx_int_param_KeyPress**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Processes key press events and calls the registered hook with the keysym

**mlx_int_param_KeyRelease**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Processes key release events and calls the registered hook with the keysym

**mlx_int_param_ButtonPress**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Processes mouse button press events with button number and coordinates

**mlx_int_param_ButtonRelease**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Processes mouse button release events with button number and coordinates

**mlx_int_param_MotionNotify**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Processes mouse motion events with pointer coordinates

**mlx_int_param_Expose**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Processes expose events, only calling the hook when all expose events are received

**mlx_int_param_generic**
- Parameters: `t_xvar *xvar`, `XEvent *ev`, `t_win_list *win`
- Description: Generic event handler for events that don't need special parameter processing

**mlx_int_param_undef**
- Parameters: None
- Description: Placeholder for undefined event types

----------------------------------------------------------------------------------------------------------------------------------------------

## String and Parsing Utilities

**mlx_int_str_str**
- Parameters: `char *str`, `char *find`, `int len`
- Returns: `int` (position or -1 if not found)
- Description: Searches for a substring within a string up to a specified length

**mlx_int_str_str_cote**
- Parameters: `char *str`, `char *find`, `int len`
- Returns: `int` (position or -1 if not found)
- Description: Searches for a substring while ignoring content within quotes

**mlx_int_str_to_wordtab**
- Parameters: `char *str`
- Returns: `char**` (array of words)
- Description: Splits a string into words separated by spaces/tabs, modifies original string by replacing separators with null terminators

----------------------------------------------------------------------------------------------------------------------------------------------

## Window Event Handling

**mlx_int_wait_first_expose**
- Parameters: `t_xvar *xvar`, `Window win`
- Description: Waits for the first expose event on a window and puts it back in the event queue

**mlx_key_hook**
- Parameters: `t_win_list *win`, `int (*funct)()`, `void *param`
- Description: Registers a callback function for key release events (simplified key event handler)

----------------------------------------------------------------------------------------------------------------------------------------------

## XPM Image Loading (Library Method)

**mlx_xpm_file_to_image**
- Parameters: `t_xvar *xvar`, `char *filename`, `int *width`, `int *height`
- Returns: `void*` (pointer to t_img)
- Description: Loads an XPM image from a file using libXpm, returns image and sets width/height

**mlx_xpm_to_image**
- Parameters: `t_xvar *xvar`, `char **data`, `int *width`, `int *height`
- Returns: `void*` (pointer to t_img)
- Description: Creates an image from XPM data array using libXpm

**mlx_int_xpm_f_image**
- Parameters: `t_xvar *xvar`, `int *width`, `int *height`, `int (*xpm_func)()`, `void *param`
- Returns: `void*` (pointer to t_img)
- Description: Internal function to create images from XPM using specified XPM loading function

**mlx_int_egal_img**
- Parameters: `XImage *img1`, `XImage *img2`
- Returns: `int` (1 if equal, 0 otherwise)
- Description: Compares two XImage structures for equality in all properties

----------------------------------------------------------------------------------------------------------------------------------------------

## Event Loop Management

**mlx_loop**
- Parameters: `t_xvar *xvar`
- Returns: `int` (0)
- Description: Main event loop that processes X events and calls registered hooks, continues while windows exist

**mlx_loop_end**
- Parameters: `t_xvar *xvar`
- Returns: `int` (1)
- Description: Signals the main loop to end by setting the end_loop flag

**mlx_loop_hook**
- Parameters: `t_xvar *xvar`, `int (*funct)()`, `void *param`
- Description: Registers a function to be called repeatedly when no events are pending

----------------------------------------------------------------------------------------------------------------------------------------------

## Mouse Functions

**mlx_mouse_move**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `int x`, `int y`
- Returns: `int` (0)
- Description: Moves the mouse pointer to specified coordinates within a window

**mlx_mouse_hide**
- Parameters: `t_xvar *xvar`, `t_win_list *win`
- Description: Hides the mouse cursor in the specified window by creating an invisible cursor

**mlx_mouse_show**
- Parameters: `t_xvar *xvar`, `t_win_list *win`
- Description: Shows the mouse cursor in the specified window (restores default cursor)

**mlx_mouse_get_pos**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `int *win_x_return`, `int *win_y_return`
- Returns: `int` (success status)
- Description: Queries the current mouse pointer position relative to the window origin, returns 0 if pointer is not on the same screen

**mlx_mouse_hook**
- Parameters: `t_win_list *win`, `int (*funct)()`, `void *param`
- Description: Registers a callback function for mouse button press events

----------------------------------------------------------------------------------------------------------------------------------------------

## Image Creation

**mlx_new_image**
- Parameters: `t_xvar *xvar`, `int width`, `int height`
- Returns: `void*` (pointer to t_img)
- Description: Creates a new image using shared memory if available, otherwise uses standard XImage

**mlx_new_image2**
- Parameters: `t_xvar *xvar`, `int width`, `int height`
- Returns: `void*` (pointer to t_img)
- Description: Creates a new image with XYPixmap format instead of ZPixmap

**mlx_int_new_xshm_image**
- Parameters: `t_xvar *xvar`, `int width`, `int height`, `int format`
- Returns: `void*` (pointer to t_img)
- Description: Creates an image using X Shared Memory extension for faster rendering

**mlx_int_new_image**
- Parameters: `t_xvar *xvar`, `int width`, `int height`, `int format`
- Returns: `void*` (pointer to t_img)
- Description: Creates a standard XImage without shared memory

----------------------------------------------------------------------------------------------------------------------------------------------

## Window Creation

**mlx_new_window**
- Parameters: `t_xvar *xvar`, `int size_x`, `int size_y`, `char *title`
- Returns: `void*` (pointer to t_win_list)
- Description: Creates a new window with specified dimensions and title, initializes graphics context and event handling

----------------------------------------------------------------------------------------------------------------------------------------------

## Drawing Functions

**mlx_pixel_put**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `int x`, `int y`, `int color`
- Description: Draws a single pixel at specified coordinates with given color directly to window

**mlx_put_image_to_window**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `t_img *img`, `int x`, `int y`
- Description: Renders an image to a window at specified coordinates, handles both shared memory and standard images

----------------------------------------------------------------------------------------------------------------------------------------------

## Color Name Database

**mlx_col_name** (data structure)
- Description: Array of color name to RGB value mappings for X11 standard color names (e.g., "red", "blue", "ghost white", etc.)

----------------------------------------------------------------------------------------------------------------------------------------------

## Screen Information

**mlx_get_screen_size**
- Parameters: `void *mlx_ptr`, `int *sizex`, `int *sizey`
- Description: Retrieves the screen dimensions and stores them in the provided pointers

----------------------------------------------------------------------------------------------------------------------------------------------

## Font Management

**mlx_set_font**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `char *name`
- Description: Sets the font to be used by mlx_string_put (only fixed-width bitmap fonts supported, use xfontsel to get valid names)

----------------------------------------------------------------------------------------------------------------------------------------------

## Text Drawing

**mlx_string_put**
- Parameters: `t_xvar *xvar`, `t_win_list *win`, `int x`, `int y`, `int color`, `char *string`
- Description: Draws a text string at specified coordinates with given color

----------------------------------------------------------------------------------------------------------------------------------------------

## XPM Parsing (Custom Parser)

**mlx_xpm_file_to_image**
- Parameters: `t_xvar *xvar`, `char *file`, `int *width`, `int *height`
- Returns: `void*` (pointer to t_img)
- Description: Loads an XPM image from a file using custom parser, memory-maps file for reading

**mlx_xpm_to_image**
- Parameters: `t_xvar *xvar`, `char **xpm_data`, `int *width`, `int *height`
- Returns: `void*` (pointer to t_img)
- Description: Creates an image from XPM data array using custom parser

**mlx_int_parse_xpm**
- Parameters: `t_xvar *xvar`, `void *info`, `int info_size`, `char *(*f)()`
- Returns: `void*` (pointer to t_img)
- Description: Core XPM parser that processes XPM format and creates an image, supports transparency

**mlx_int_get_line**
- Parameters: `char *ptr`, `int *pos`, `int size`
- Returns: `char*` (line content)
- Description: Extracts a line from XPM file data between quotes

**mlx_int_static_line**
- Parameters: `char **xpm_data`, `int *pos`, `int size`
- Returns: `char*` (line content)
- Description: Retrieves a line from static XPM data array

**mlx_int_get_col_name**
- Parameters: `char *str`, `int size`
- Returns: `int` (color identifier)
- Description: Converts character sequence to a unique color identifier number

**mlx_int_get_text_rgb**
- Parameters: `char *name`, `char *end`
- Returns: `int` (RGB color value)
- Description: Converts color name or hex string to RGB value, supports X11 color names

**mlx_int_xpm_set_pixel**
- Parameters: `t_img *img`, `char *data`, `int opp`, `int col`, `int x`
- Description: Sets a pixel in image data accounting for byte order and bytes per pixel

**mlx_int_file_get_rid_comment**
- Parameters: `char *ptr`, `int size`
- Description: Removes C-style comments (/* */ and //) from XPM file content by replacing with spaces
